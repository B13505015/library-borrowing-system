# Feature roadmap (requested updates)

已先完成資料庫層基礎：
- 借閱上限規則：`loan_policies`
- 預約系統（含優先序）：`reservations`
- 收藏功能：`favorites`
- 逾期警示/罰款：`user_penalties`
- 停權管理：`user_suspensions`
- 書評系統：`reviews`

接續建議（API/UI）：
1. API：新增 reservation / favorites / reviews / suspend user endpoints。
2. Admin UI：新增使用者借閱詳情按鈕、停權按鈕、主題借閱熱度圖表。
3. User UI：顯示總書籍、借出中、可借數量；把空狀態區塊移到底部。
4. Login UI：管理者登入入口移至右上角。
